## 项目总览 (Overview)
* 项目名称: 家鲜记 (Home Fresh Record)
* 目标: 开发一个极简的、主要在手机上操作的家庭物品（主要是食品和日用品）保质期管理应用。
* 核心理念: 极简录入、被动提醒、即用即走。砍掉一切非核心功能，降低用户维护列表的门槛。

### 核心功能描述 (Functionality)
#### 极简物品录入 (Input Module):

必填核心项: “物品名称”和“过期日期”。

录入模式:
1. 拍照录入：拍摄照片，自动识别物品名、生产日期、保质期
2. 手动模式: 手动输入物品名，通过日历选择过期日期。

#### 快捷模式:

保质期倒推: 提供“生产日期 + 保质期（天/月）”计算器，自动算出过期日期。

常用模板: 内置常用物品（如牛奶：7天）模板，点击快速填表。

### 直观列表展示 (List Module):

* 状态分类: 物品按状态分为已过期（红色）、临期中（黄色，默认提前3天）、安全（绿色）。

* 智能排序: 列表强制按“距离过期天数”升序排列，最紧急的永远在最上面。一眼即可掌握风险。

### 物品状态管理 (Status Management):

* 一键消耗: 提供极简操作（如点击勾选框、左滑），标记物品为“吃掉了/用完了”。被标记的物品移出活跃列表，不删数据以备复用。

* 一键复用: 对于历史消耗物品，提供“再来一个”功能，可直接重新启用，仅需修改过期时间。


### 技术方案
* 平台: PWA（渐进式Web应用），优化移动端体验，实现全屏运行和离线可用。

* 技术栈:
前端框架: Next.js (或者 Vite + React/Vue)
样式库: Tailwind CSS
部署平台：cloudflare worker
数据库：D1
